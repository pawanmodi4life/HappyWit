# HappyWit Website

Premium React + Vite + Tailwind website for HappyWit.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Framework: Vite
4. Build Command: `npm run build`
5. Output Directory: `dist`

## Important

The team photos currently use LinkedIn image URLs. For production, download photos and place them in `public/team/`, then update the image paths in `src/App.jsx`.
